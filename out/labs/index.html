MANUTD France

See it ONLINE ↑

Manchester United France is the official association for the French MU supporters.
Their website needed to be rebuilt. This project had several components that needed to interact with each other: website, forum, blog, online subscription system, ticket service for MU matches, and an admin section to manage the whole website.
The final architecture is built with a mix between wordpress-base website and blog, PHP custom code for the management system, and PHPBB to power the forum.

Design:
Photoshop
Coded under:
Netbeans IDE
Architecture made of:
Wordpress, PhpBB, OOPHP

Car-sharing application

Currently OFFLINE ↑

This car-sharing application was an attempt to reduce the cost of going home for students:
some students have a car and want to share costs, others don't want to pay the full price of a train ticket. That's exactly where this app came into play.
The car-sharing application is powered by PHP on the server, without any framework. On the client side, it makes heavy use of JavaScript and interacts with the server asynchronously, for a fast user experience.

Design:
Photoshop & Illustrator
Coded under:
Netbeans IDE
Based on:
Custom PHP & AJAX interactions

Georges Duquin

Georges Duquin is a French author and painter.
